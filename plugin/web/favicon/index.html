网站Favicon获取
说明

Favicon获取接口：

https://favicon.cccyun.cc/要获取的网址

例如：

https://favicon.cccyun.cc/http://tool.cccyun.cc/

注：接口有来源校验，空来源将被禁止访问

如果网站变更了favicon，可以点击清除缓存，然后点获取，在跳转后的页面按Ctrl+F5即可显示最新的图标
